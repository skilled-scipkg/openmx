## Input file for the restart calculation

An input file '*System.Name*.dat#' is generated at every MD step for
the restart calculation with the final structure and the same
'Grid_Origin' explained in the Section 'Fixing the relative position
of regular grid'.
Using the file '*System.Name*.dat#', it can be possible to continue MD calculations
and geometry optimization from the last step.
