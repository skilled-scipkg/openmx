# OpenMX Forum

For discussion of technical issues on OpenMX and ADPACK,
there is a forum (http://www.openmx-square.org/forum/patio.cgi).
It is expected that the forum is utilized
for sharing tips in use of OpenMX and for further code
development. Points of concern for use of this forum
can be found in http://www.openmx-square.org/forum/note.html
