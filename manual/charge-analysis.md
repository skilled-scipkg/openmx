# Charge analysis

Although it is a somewhat ambiguous issue to assign effective charge
to each atom, OpenMX provides three schemes, Mulliken charge analysis,
Voronoi charge analysis, and electro-static potential (ESP) fitting method,
to analyze the charge state of each atom.
