## Compilation of jx

To compile 'jx' which is a post-processing code calculating $J_{ij}$ from the OpenMX output,
please move to the directory 'source', and type as follows:

```text
                                                                                    
     % make jx
```

When the compilation is completed normally, then you can find an executable file 'jx' in the directory 'work'.
